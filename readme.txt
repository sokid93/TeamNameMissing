Cuatro

Cuatro es un juego matemático de partida infinita donde usamos las raíces cuadradas. El jugador debe conseguir el 
número cuadrado perfecto más alto, mientras se enfrenta a las adversidades del table

This is a demo made for the Global Game Off Jam 2023.

This demo was created using Unreal Engine 4.27.

The game and most graphical assets are freely available within the Unreal Engine. The music and sfx are available
under a CC license.
